## 阅读顺序

建议按下面顺序查阅：

1. 先看 [../apps/docs/spec/overview.md](../apps/docs/spec/overview.md)，建立整体产品与页面级认知
2. 再看 [../apps/docs/spec/tokens.md](../apps/docs/spec/tokens.md) 和 [../apps/docs/spec/components.md](../apps/docs/spec/components.md)，确认数值规则与组件边界
3. 需要出图时查看 [guides/pencil-guide.md](./guides/pencil-guide.md)
4. 需要统一内容样例时查看 [fixtures/markdown-fixture.md](./fixtures/markdown-fixture.md)
