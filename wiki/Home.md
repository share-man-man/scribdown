# Scribdown Wiki

Scribdown 是一个基于 Monorepo 的多端 Markdown 渲染项目，当前包含浏览器插件与 VS Code 插件两个应用，以及共享渲染与 UI 能力包。

## 一分钟启动

```bash
pnpm install
pnpm dev
```
