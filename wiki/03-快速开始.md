# 快速开始

## 环境准备

- Node.js `24.15.0`
- pnpm `10.33.0`

## 安装依赖

```bash
pnpm install
```

## 本地开发

```bash
# 启动 packages/tools 的 watch
pnpm run dev:packages

# 启动 apps 的开发进程
pnpm run dev:apps

# 一键启动（默认命令）
pnpm run dev
```

## 构建与校验

```bash
pnpm run build
pnpm run lint
pnpm run typecheck
pnpm run test
pnpm run e2e
```

## 常见场景命令

| 场景 | 命令 |
| --- | --- |
| 创建版本变更记录 | `pnpm run changeset` |
| 计算并写入版本号 | `pnpm run version-packages` |
| 发布包 | `pnpm run release` |
| 清理构建产物与依赖目录 | `pnpm run clean` |

## 开发建议

1. 先在 `packages/*` 完成可复用能力，再在 `apps/*` 接入。
2. 变更公共 API 时同步补充测试与 Wiki。
3. 发布前至少执行一次 `build + test + typecheck`。
