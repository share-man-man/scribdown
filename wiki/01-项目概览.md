# 项目概览

## 定位

Scribdown 目标是提供统一的 Markdown 渲染体验，覆盖浏览器插件与 VS Code 插件，并复用同一套渲染核心与视觉组件。

## 当前能力

- 浏览器端：基于 React + Vite 的插件弹窗预览
- VS Code 端：注册 `scribdown.openPreview` 命令并展示 Webview
- 渲染层：`remark + rehype` 管线，支持 HTML 清洗与代码高亮
- 共享层：统一常量、枚举与主题约定

## 技术栈

- 语言与类型：TypeScript
- 前端：React 19、Vite 7
- Markdown 渲染：unified、remark、rehype
- 安全清洗：rehype-sanitize、DOMPurify
- 高亮：Shiki
- 工程化：pnpm workspace、Turborepo、Changesets
- 测试：Vitest、Playwright
